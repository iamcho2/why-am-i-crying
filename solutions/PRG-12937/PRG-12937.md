![image](https://user-images.githubusercontent.com/28949235/122722404-a33fb000-d2ac-11eb-983c-66b1ed60fa2b.png)

### 문제 정리

짝수 - "Even" 반환, 홀수 - "Odd" 반환

### 생각해보기

2로 나눠서 나머지로 분기

### 코드 쓰기

### 내 풀이

```c++
#include <string>
#include <vector>

using namespace std;

string solution(int num) {
    string answer = "";
    
    if (num % 2 == 0){
        answer = "Even";
    } else {
        answer = "Odd";
    }
    
    return answer;
}
```
