### 문제 정리

두 배열의 1~n번째 요소들을 서로 곱한 후에 다 더하기

### 생각해보기

a, b의 길이가 항상 같으므로 하나 길이 구해서  
그만큼 for문 돌려서 sum 변수에 다 더하면 될 듯  

### 코드 쓰기

벡터의 사이즈 리턴 함수 -> a.size()

### 내 풀이

```c++
#include <string>
#include <vector>

using namespace std;

int solution(vector<int> a, vector<int> b) {
    int answer = 0;
    int size = a.size();
    
    for(int i=0; i<size; i++) {
        answer += a[i]*b[i];
    }
    
    return answer;
}
```
