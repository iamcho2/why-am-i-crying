# #12912 두 정수 사이의 합

![image](https://user-images.githubusercontent.com/28949235/122742318-c83e1e00-d2c0-11eb-973d-c58d37d5e0d3.png)

### 문제 정리

a ~ b 사이 정수 다 더한 수 리턴

### 생각해보기

a 에서 b까지 for문 돌리면 될듯? 대소 비교 후에!!

### 코드 쓰기

### 내 풀이

```c++
#include <string>
#include <vector>

using namespace std;

long long solution(int a, int b) {
    long long answer = 0;
    
    if(a>b){
        int t = a;
        a = b;
        b = t;
    }
    
    for(int i=a; i<=b; i++){
        answer += i;
    }
    
    return answer;
}
```
