### **문제 정리**

소 n번관찰 - 소의 번호, 소의 위치로 이루어짐
 소를 10마리 가지고 있음 → 소의 번호는 1이상 10이하
 소의 위치 : 길의 왼0 오1
 같은 번호의 소가 위치를 바꾼 게 몇 번인지 세면 됨

### **생각해보기**

10칸 배열 만들어서 번호 바뀔때마다 1 더해주면 됨

### **코드 쓰기**

### **내 풀이**

```swift
let N = Int(readLine()!)!
var arr = Array(repeating: -1, count: 11)
var answer = 0

for _ in 0..<N {
    let cow = readLine()!.split(separator: " ").map { Int(String($0))! }
    if arr[cow[0]] == -1 {
        arr[cow[0]] = cow[1]
    } else {
        if arr[cow[0]] != cow[1] {
            answer += 1
        }
        arr[cow[0]] = cow[1]
    }
}

print(answer)
```
