### 문제 정리

문자열 s를 숫자로 변환한 결과를 반환

### 생각해보기

걍 stoi 쓰는건가? 부호가 있어도 stoi쓸 수 있나??

### 코드 쓰기

### 내 풀이

```c++
#include <string>
#include <vector>

using namespace std;

int solution(string s) {
    int answer = 0;
    answer = stoi(s);
    return answer;
}
```
