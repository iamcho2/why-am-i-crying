### 문제 정리

for문 돌리기

### 생각해보기

### 코드 쓰기

2중 for문

### 내 풀이

```c++
#include <iostream>

using namespace std;

int main(void) {
    int n;
    int m;
    cin >> n >> m;
    
    for(int i=0; i<m; i++){
        for(int k=0; k<n; k++){
            cout << "*";
        }
        cout << endl;
    }
    
    return 0;
}
```
